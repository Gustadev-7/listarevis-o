Linguagens importantes para o desenvolvimento web

HTML

HTML é uma linguagem de marcação (não de programação) que estrutura o conteúdo das páginas web através de elementos e tags. Foi criada por Tim Berners-Lee em 1991 e é mantida pelo World Wide Web Consortium (W3C). Com ele é possível a formatação de elementos como:

  • Textos
  • Imagens
  • Formulários
  • Tabelas

CSS

CSS (Cascading Style Sheets) é a linguagem que controla a apresentação visual de documentos HTML e XML. Foi criada em 1996 por Håkon Wium Lie para resolver o problema de separação entre conteúdo (HTML) e apresentação.

JavaScript

O JavaScript (JS) é uma linguagem de programação leve, interpretada e orientada a objetos que é amplamente utilizada para criar interatividade em páginas web.
